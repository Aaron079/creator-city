# Creator City 第一版已知限制

**版本：** 第一版（2026-06-16）  
**更新人：** P2-FIRST-USERS 任务

---

## 存储 / 保存

| 限制 | 说明 | 计划 |
|---|---|---|
| 自动云端保存未开启 | 画布内容实时保存到本地浏览器（localStorage），需手动点击「保存」按钮同步到云端 | 后续版本开启自动保存 |
| 清除浏览器缓存会丢失本地草稿 | localStorage 草稿未同步云端时如清除缓存将丢失 | 建议用户养成手动保存习惯 |

---

## 计费 / 支付

| 限制 | 说明 | 计划 |
|---|---|---|
| 平台积分充值暂不开放 | 普通用户无法充值积分，也无法用平台积分生成内容 | P2-BILLING（后续版本） |
| 无自动退款 | 会员费一经开通不退款，特殊情况需联系管理员 | 后续版本 |
| 无自动续费 | 会员到期需重新申请，管理员人工审核 | 后续版本 |
| 会员审批人工处理 | 通常 1 个工作日内，非即时开通 | 长期 SOP |

---

## AI 生成 / Provider

| 限制 | 说明 | 计划 |
|---|---|---|
| 视频 BYOK（Seedance）生成接入中 | 凭证可预先保存，但当前无法用 BYOK 模式生成视频 | 接入中，即将上线 |
| 图片 Provider 仅支持 Seedream（火山方舟） | OpenAI Image DALL-E 等暂未接入 | 后续版本 |
| 文本 Provider 支持 DeepSeek / Kimi / OpenAI | 阿里通义 / 百度千帆 / Gemini 等暂未接入 | 后续版本 |
| 平台积分生成不对外开放 | 仅 BYOK 模式，无平台代付 | 后续版本 |
| 火山方舟 Seedream 不支持自动连接测试 | 需在画布实际生成验证 | Provider 限制，无法绕过 |

---

## Marketplace

| 限制 | 说明 | 计划 |
|---|---|---|
| 仅为展示与意向登记 | Marketplace 不触发授权成交、资金托管、积分结算 | 后续版本开放担保交易 |
| 不开放平台担保交易 | 合作意向仅记录双方沟通意向 | P2-BILLING 后 |
| 需要会员才能提交意向 | 非会员无法联系创作者或提交意向 | 设计如此，不变 |
| 无自动授权凭证 | 意向 RESPONDED 后不自动生成 LicenseGrant | 后续版本 |

---

## 画布功能

| 限制 | 说明 | 计划 |
|---|---|---|
| 分镜 / 选角 / StyleBible 工具已移除 | 第一版精简为最小稳定生成链路 | 后续版本重新集成 |
| 高级编辑（image-editor）已移除 | 第一版不含图片二次编辑 | 后续版本 |
| 网络切换可能导致保存失败 | 在移动网络或 VPN 切换时 PUT canvas 可能失败 | 已有本地草稿保护，失败不丢数据 |

---

## 其他

| 限制 | 说明 | 计划 |
|---|---|---|
| NFT / wallet / on-chain 不支持 | 第一版不做区块链相关功能 | 长期后续 |
| 需求广场 / 报价方案 / 阶段交付 / 托管结算 | 预览页面，尚未开放 | 第二版 |
| 创作者主页 | 预览中 | 第二版 |
| 首批用户为邀请 / 试用阶段 | 非公开注册，需受邀 | 稳定后开放 |
