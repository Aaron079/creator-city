# Creator City 首批用户试用指南

**版本：** 第一版（2026-06-16）  
**适用对象：** 受邀首批试用用户  
**试用模式：** 会员订阅 + 自带 API Key（BYOK）

---

## 试用目标

Creator City 是 AI 导演工作台。你可以用自己的 AI Provider API Key，在画布上创作图片和文本内容，并在 Marketplace 发布或浏览创作者作品。

**第一版开放能力：**
- AI 画布创作（图片 / 文本节点，BYOK 模式）
- 资产库管理
- Marketplace 展示 + 授权合作意向
- 会员订阅开通

**第一版暂不开放：**
- 平台积分充值与消费
- 自动退款
- 视频 BYOK（Seedance，凭证可预存）
- 自动云端保存（当前为本地草稿 + 手动保存）

---

## 开始步骤

### 第一步：注册 / 登录

1. 打开 Creator City 网站
2. 点击右上角「注册」
3. 填写邮箱、密码、显示名称
4. 完成注册后系统自动登录

如已有账号，直接「登录」即可。

---

### 第二步：开通会员

Creator City 第一版采用会员订阅模式。

**费用：** ¥100 / 月（平台服务费，不含第三方 AI API 调用成本）

**开通流程：**
1. 登录后，点击右上角用户名 → **会员中心**
2. 或直接访问 `/account/membership`
3. 阅读会员权益说明
4. 提交开通申请，按页面说明完成线下转账（填写指定备注）
5. 管理员人工核实后通常在 1 个工作日内开通
6. 开通后有效期 30 天

> **注意：** 会员费是平台服务费，**不包含第三方 AI API 调用成本**（DeepSeek / 火山方舟 / OpenAI 等的 API 费用由你直接支付给服务商）。

---

### 第三步：添加 API Key（BYOK）

第一版采用 BYOK 模式，你需要准备自己的 Provider API Key 才能在画布生成内容。

**目前已支持的 Provider：**

| Provider | 类型 | 状态 |
|---|---|---|
| DeepSeek V4 Flash / Pro | 文本 | ✅ 已上线 |
| Kimi K2 | 文本 | ✅ 已上线 |
| OpenAI GPT | 文本 | ✅ 已上线 |
| Seedream Image（火山方舟）| 图片 | ✅ 已上线 |
| Seedance Video（火山方舟）| 视频 | 🟡 凭证可存，生成接入中 |

**添加步骤：**
1. 登录 Provider 控制台，创建 API Key（以 `sk-` 等格式开头）
2. 访问 Creator City → 右上角 → **我的 API 账户**
3. 点击「添加 API 账户」，填写 Provider、账户名称、API Key
4. 保存后仅显示末 4 位，Key 加密存储

**重要：** API Key 是 Provider 控制台生成的访问密钥，**不是你的网页登录邮箱和密码**。

详细步骤见 → [`/help/api-keys`](https://creator-city-vert.vercel.app/help/api-keys)

---

### 第四步：进入画布

1. 点击顶部导航「创作」→「AI 画布」，或访问 `/create` 或 `/create-v2`
2. 系统自动为你创建或恢复项目
3. 在画布上拖拽添加节点（图片节点 / 文本节点）
4. 在节点对话框中填写 Prompt，选择「我的 API 账户」，点击「生成」

---

### 第五步：保存作品

**当前保存机制：**
- 画布内容**实时保存到本地浏览器**（刷新也不会丢失，但清除浏览器缓存会丢失）
- 点击画布右上角「保存」按钮可**手动同步到云端**
- 云端保存成功后在其他设备也可恢复

> 当前不支持自动云端保存，请养成定期手动保存的习惯。

---

### 第六步：管理资产库

- 生成成功的图片 / 视频 / 文本会自动出现在「资产库」(`/assets`)
- 可在资产详情页发布到 Marketplace（需要会员）
- 资产可在多个画布项目中复用

---

### 第七步：浏览 Marketplace / 提交授权合作意向

1. 访问 `/marketplace` 浏览创作者展示的作品
2. 点击作品详情 → 查看 Listing 信息
3. 会员可提交授权合作意向（非正式成交，只是记录合作意向）

> **注意：** Marketplace 第一版仅为展示与意向登记，**不代表授权成交**，不触发平台担保、资金托管或积分结算。合作意向仅用于双方沟通记录。

## 常见问题

**Q: 我可以不用 API Key，用平台积分生成吗？**  
A: 第一版不对普通用户开放平台积分充值和消费。你需要自带 Provider API Key。

**Q: API Key 安全吗？**  
A: Key 使用 AES-256-GCM 加密存储，保存后只显示末 4 位，Creator City 无法查看完整 Key。如怀疑泄漏，请立即在 Provider 控制台吊销。

**Q: 生成的内容版权归谁？**  
A: 你使用自己的 API Key 生成的内容，版权归你所有（受 Provider 条款约束）。Marketplace 展示不等于授权转让。

**Q: 会员费能退款吗？**  
A: 会员费一经管理员审批开通后不退款。如有疑问请在提交前联系管理员确认。
